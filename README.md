# 항공편 지연 예측 


---

#### 개인 프로젝트

#### 분석 기간 : 2023.05.10 - 2023.05.14

- BDAC 학회 최종 리포트

---

## 방법

1. 100만 건 중 예측하고자 하는 지연 여부 레이블에 다수의 결측값이 존재함을 확인.
2. 결측값이 없는 데이터를 사용하여 초기 모델 학습.

#### EDA 및 전처리
- 종속변수와 다른 변수 간의 상관관계 분석
- 불필요한 변수 제거
- 결측치 처리
- 범주형 변수 변환
- 스케일링

#### 모델링
- Decision Tree, Random Forest, LightGBM 등

#### 파라미터 튜닝
- Grid Search

## 최종 모델
- LightGBM
